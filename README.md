# 🔍 QR Safety Scanner
🔐 Ensuring safer browsing through QR intelligence

## 📌 Project Overview
The QR Safety Scanner is a security-focused application that scans QR codes and analyzes the embedded URLs to detect potential threats. It helps users identify suspicious, unsafe, or malicious links before opening them.

## 🚀 Features
- Scan QR codes in real-time
- Detect suspicious or unsafe URLs
- Risk score generation
- Displays protocol and destination info
- Alerts users about potential threats
- Simple and user-friendly interface

## 🛠️ Technologies Used
- JavaScript / Python
- React Native / Expo Go
- QR Scanner Libraries
- Web APIs for URL analysis

## 📂 Project Structure
- `src/` → Main application code
- `components/` → UI components
- `utils/` → URL analysis logic
- `App.js` → Entry point

## 🔗 Demo / Output
Example output:
- Risk Score: 50/100
- Protocol: HTTP (Unsecure)
- Status: Suspicious
